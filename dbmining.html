DB Mining

In the DB-Mining project, we aim to investigate how databases and ORM are used in end-user open-source applications. We focused on active GitHub projects with at least 1,000 starts, 1,000 commits, 10 collaborators, and Java as a primary language. We use an infrastructure that automatically clones the projects, runs the heuristics over each project using git grep, and populates a database with the obtained outputs. Finally, we manually validated the results provided by the heuristics execution. The complete experimental package, containing the data and scripts used in this research, and the reproducibility instructions, is available at GitHub
